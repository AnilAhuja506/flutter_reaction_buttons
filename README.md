# 🎭 Reactions Button in Flutter (LinkedIn & Facebook Style)  

This project implements a **reactions button in Flutter**, similar to **LinkedIn and Facebook**, featuring hover effects, long-press interactions, and smooth transition animations on tap.  

## ✨ Features  

- 🎭 **LinkedIn & Facebook-style Reactions** – Supports multiple reaction options.  
- 🖱️ **Hover Effect (Web & Desktop)** – Displays reaction options when hovered.  
- 📌 **Long-Press Support (Mobile)** – Users can trigger reactions by long-pressing.  
- ⚡ **Smooth Transition Animation** – Ensures a fluid user experience on tap.  
- 🎨 **Optimized UI** – Designed for responsiveness and seamless interaction.  
- 🌍 **Cross-Platform Support** – Works on **Android, iOS, Web, and Desktop**.
